API Examples Using Fetch

Example 1: TVmaze API

This example demonstrates how to use fetch to get data from the TVmaze API. Enter a show name below to search for TV shows.

How does this work?

The fetch function is used to make HTTP requests in JavaScript. In this example, we use fetch to call the TVmaze API's search endpoint:
https://api.tvmaze.com/search/shows?q=QUERY
The API returns a list of shows matching the search term. We then display the results below.

Example 2: Text-to-Speech (SpeechSynthesis API)

This example shows how to use the browser's built-in SpeechSynthesis API to convert text to speech. Enter some text and click the button to hear it spoken aloud.

How does this work?

The SpeechSynthesis API is built into most modern browsers. When you click the button, JavaScript creates a SpeechSynthesisUtterance object with your text and calls speechSynthesis.speak() to play the voice.
This is a great way to add accessibility or interactive features to your web apps without needing an external API key.
